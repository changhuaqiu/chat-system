## 2.2 数据流

### 2.2.1 消息发送流程

```
用户输入 → ChatPage → socket.emit('sendMessage')
                                │
                                ▼
                    server.js: socket.on('sendMessage')
                                │
                                ▼
                    eventBus.publish('message.created')
                                │
                    ┌───────────┴───────────┐
                    ▼                       ▼
            DB 持久化                  事件监听器
            (messages表)                     │
                                            ▼
                                    botService.onMessageCreated
                                            │
                                            ▼
                                    检查触发条件
                                    (自由模式/提及模式)
                                            │
                                            ▼
                                    botRuntime.generateResponse
                                            │
                                            ▼
                                    LlmAdaptor.chat()
                                            │
                                            ▼
                                    LLM API 调用
                                            │
                                            ▼
                                    eventBus.publish('message.created')
                                            │
                                            ▼
                                    io.to(room).emit('messageReceived')
```

## 2.3 核心组件职责

| 组件 | 文件 | 职责 |
|------|------|------|
| server.js | backend/src/server.js | HTTP/WebSocket 服务入口，初始化 |
| eventBus | backend/src/services/eventBus.js | 事件发布/订阅，持久化事件 |
| botService | backend/src/services/botService.js | 消息处理，Bot 触发调度 |
| botRuntime | backend/src/services/botRuntime.js | Bot 运行时，管理适配器 |
| agentService | backend/src/services/agentService.js | OpenClaw Agent 管理 |
| oneApiService | backend/src/services/OneApiService.js | One-API 集成，渠道管理 |
| agent-router.js | backend/src/agent-router.js | Agent 消息路由 |

## 2.4 技术选型理由

| 技术 | 选型理由 |
|------|----------|
| Fastify | 高性能 HTTP 框架，插件化架构 |
| Socket.io | 成熟的 WebSocket 实现，自动重连 |
| SQLite | 轻量级，适合中小规模，零配置 |
| React | 组件化，生态丰富 |
| Vite | 快速开发，HMR 支持 |
| TailwindCSS | 原子化 CSS，快速开发 UI |
